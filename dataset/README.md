## Dataset

Kaggle Skin Wound Segmentation Dataset

- Training Images: 2208
- Training Masks: 2208
- Testing Images: 552
- Testing Masks: 552

Only a small representative sample/subset (20 images and 10 masks) is included in this repository due to GitHub storage and repository-size limitations.

Full Dataset: https://drive.google.com/drive/folders/1wLKH-nhexQZvK_Ka2QQR32B0Qpk_JWQn?usp=sharing
